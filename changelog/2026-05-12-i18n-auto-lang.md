# GitHub Pages 自动中英文切换

## 变更内容

### 新增功能

- **浏览器语言自动检测**：根据用户浏览器语言自动选择中文或英文显示
- **语言手动切换**：在导航栏添加语言切换按钮，用户可手动选择语言
- **本地化持久化**：语言选择保存在 localStorage，下次访问自动应用
- **URL 参数支持**：支持通过 `?lang=zh` 或 `?lang=en` 指定语言版本

### 优先级

1. URL 参数（`?lang=zh/en`）- 最高优先级
2. localStorage 存储的用户偏好
3. 浏览器语言自动检测
4. 默认中文

## 使用说明

### 访问默认语言

直接访问 `https://lessup.github.io/meta-human/`，系统会自动检测浏览器语言。

### 访问指定语言

- 中文：`https://lessup.github.io/meta-human/?lang=zh`
- 英文：`https://lessup.github.io/meta-human/?lang=en`

### 手动切换

点击导航栏右上角的语言切换按钮（显示 "EN" 或 "中文"）即可切换语言。

## 浏览器兼容性

支持所有现代浏览器（Chrome、Edge、Firefox、Safari），语言检测使用标准 `navigator.language` API。
